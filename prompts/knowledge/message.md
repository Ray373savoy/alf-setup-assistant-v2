# Message Node（メッセージ送信ノード）

単純なテキストメッセージを送信するノードです。（print関数と考えれば分かりやすいです）

## 提案書のポリシー案内活用
タスク提案書にポリシー案内メッセージがあれば**そのまま**使用します。
- 提案書のポリシーテーブル → `blocks[].value`に挿入
- 任意に修正せず原文そのまま使用

## 特徴

- **単純テキストメッセージ送信**
- **メモリ変数活用不可**
- パーソナライズ変数`#{名前}`の使用は可能だが、context参照のため非会員では存在しない場合あり

## いつ使う？

- 固定された案内文言（変動のないメッセージ）
- 画像添付
- ボタン提供

**変数活用が必要な場合はエージェントノードの使用を推奨**

## JSON構造

```json
{
  "type": "message",
  "message": {
    "blocks": [
      {
        "type": "text",
        "value": "ご案内メッセージです。"
      }
    ],
    "buttons": null,
    "files": null
  }
}
```

## エージェントノード vs メッセージノード

| 状況 | ノード選択 |
|------|----------|
| 動的対話、知識基盤の案内 | エージェントノード |
| 固定メッセージ、画像、ボタン | メッセージノード |
| メモリ変数活用が必要 | エージェントノード |
| パーソナライズ変数のみ必要 | メッセージノード |

## 実際の例

### 画像添付案内

```json
{
  "type": "message",
  "message": {
    "blocks": [
      {
        "type": "text",
        "value": "{案内タイトル}です。\n\n下記の画像を参考にして進めてください。"
      }
    ],
    "buttons": null,
    "files": null
  }
}
```

### サービス利用不可案内

```json
{
  "type": "message",
  "message": {
    "blocks": [
      {
        "type": "text",
        "value": "現在、該当{条件}では{サービス名}をご利用いただけません。\n{代替案内メッセージ}"
      }
    ],
    "buttons": null,
    "files": null
  }
}
```

### エラー案内（相談員接続前）

コードノードでエラー発生時やステータス判定で対応不可の場合、相談員接続前の顧客案内用。
🔴 **「処理中にエラーが発生しました」だけの汎用メッセージは禁止。** 各エラー分岐ごとに処理段階や理由を具体的に記述すること。

**API呼び出しエラーの例（処理段階を明示）：**

```json
{
  "type": "message",
  "message": {
    "blocks": [
      {
        "type": "text",
        "value": "注文情報の検索中にエラーが発生しました。\n担当者にお繋ぎいたしますので、少々お待ちください。"
      }
    ],
    "buttons": null,
    "files": null
  }
}
```

**ステータス判定による対応不可の例（理由を明示）：**

```json
{
  "type": "message",
  "message": {
    "blocks": [
      {
        "type": "text",
        "value": "現在の配送ステータスでは、お届け先の変更ができません。\n担当者にお繋ぎいたしますので、少々お待ちください。"
      }
    ],
    "buttons": null,
    "files": null
  }
}
```

**処理段階別のメッセージ例：**

| エラーの発生箇所 | メッセージ例 |
|----------------|------------|
| 注文検索 | 「注文情報の検索中にエラーが発生しました」 |
| 本人確認 | 「ご本人確認の処理中にエラーが発生しました」 |
| キャンセル実行 | 「キャンセル処理中にエラーが発生しました」 |
| 住所変更実行 | 「配送先の変更処理中にエラーが発生しました」 |
| ステータス判定（対応不可） | 「現在の配送ステータスでは変更ができません」 |
| キャンセル期限超過 | 「キャンセル可能な期限を過ぎているため、自動処理ができません」 |
